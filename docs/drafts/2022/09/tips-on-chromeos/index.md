| Shortcut               | Description            |
| ---------------------- | ---------------------- |
| `ctrl` + `shift` + `+` | Increase size of tabs. |
| `ctrl` + `shift` + `+` | Decrease size of tabs. |
| `ctrl` + `+`           | Zoom in a page.        |
| `ctrl` + `-`           | Zomm out a page.       |
