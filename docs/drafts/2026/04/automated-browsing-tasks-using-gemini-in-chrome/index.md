## Tips and Traps

1. Gemini in Chrome is not available on Linux currently!
    For more discussions,
    please refer to
    [Gemini in Chrome, for Linux](https://support.google.com/gemini/thread/412505429/gemini-in-chrome-for-linux?hl=en)
    .

2. Gemini Auto Browse introduces new features such as
    - Actionable Automation: It can automatically click, search, compare options, and add items to carts, reducing manual, repetitive tasks.
    - Cross-App Integration: Integrates with Gmail, Calendar, and Maps to gather context for tasks, such as planning trips or finding locations.
    - Security & Control: It requires user permission for sensitive actions, such as logging into accounts (using Google Password Manager) or completing purchases.
    - User Interface: A dedicated Gemini icon in the Chrome browser bar activates the tool, providing a permanent sidebar for interactions. 
    However,
    Auto Browse **requires a subscription of Google AI Pro** or Google AI Ultra.
