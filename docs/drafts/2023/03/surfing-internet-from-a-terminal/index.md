## w3m

```
sudo apt install w3m w3m-img
```

- pure text rendering and thus least dependency requirements

## [brow6el](https://codeberg.org/janantos/brow6el)

- [Brow6el](https://codeberg.org/janantos/brow6el)
  is a minimalistic graphical terminal web browser using sixels.
- You need to run it in a terminal with sexels support.
- It works best in Ubuntu/Debian current with known issues in Fedora.
  The experience is great (close to a normal browser) if everything works well.

## [browsh](https://github.com/browsh-org/browsh)

- [Browsh](https://github.com/browsh-org/browsh)
  is a fully-modern text-based browser, rendering to TTY and browsers
- The experience isn't as good as Brow6el but better than w3m.
- Requires Firefox to be installed.
